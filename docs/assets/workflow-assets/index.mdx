## Asset Categories Overview

| Category | Purpose |
|----------|---------|
| **Sources** | Define entry points for data ingestion from external systems. |
| **Sinks** | Define destinations for data output and delivery. |
| **Input Processors** | Handle the initial processing and validation of incoming data. |
| **Output Processors** | Handle the final formatting and transmission of outgoing data. |
| **Flow Processors** | Transform, route, and process data within the workflow pipeline. |
| **Formats** | Specify how messages are parsed and serialized. |
| **Connections** | Store reusable connection parameters for external services. |
| **Services** | Interface with external APIs, databases, and cloud services. |
| **Resources** | Define shared configuration and reusable components. |
| **Extensions** | Add custom functionality and macros to the platform. |
| **Workflows** | Orchestrate the flow of data through connected processors. |
